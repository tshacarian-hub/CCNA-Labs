# CCNA Lab 05 — Ethernet Frame Analysis

## Objective
Analyze Ethernet frames and understand encapsulation.

## Topology
PC0 → Switch → PC1

## Tools
Cisco Packet Tracer (Simulation Mode)

## Steps
1. Built LAN network
2. Assigned IPv4 addresses
3. Sent ICMP traffic using ping
4. Inspected packet structure in simulation mode

## Ethernet Frame Fields
- Source MAC
- Destination MAC
- Type (IPv4)
- Frame Check Sequence (FCS)

## Key Concept
Encapsulation and de-encapsulation across OSI layers.

## Result
Successfully analyzed packet flow and Ethernet frame structure.
