## Lab Objectives

Understand the role of basic network devices:

- Router
- Switch
- Client
- Server

## Device Roles

Router  
Routes traffic between networks.

Switch  
Connects devices inside a LAN.

Client  
A device that requests services from a server.

Server  
Provides services to clients.
