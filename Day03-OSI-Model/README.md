# CCNA Lab 03 — OSI Model Demonstration

This lab demonstrates how devices communicate using the OSI model.

## Topology
<img width="664" height="951" alt="Network topology" src="https://github.com/user-attachments/assets/09f58fcd-0cf5-49f7-8ecb-a565882f2977" />

PC0 → Switch → PC1

## Tools Used
- Cisco Packet Tracer

## Steps
1. Built a small LAN
2. Assigned IPv4 addresses
3. Tested connectivity with ping
4. Observed packet flow using simulation mode

## OSI Layers Observed
Layer 1 – Physical
Layer 2 – Data Link
Layer 3 – Network
Layer 4 – Transport
Layer 7 – Application

## Result
Successful ICMP communication between two hosts.
