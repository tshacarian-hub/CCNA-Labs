# CCNA Lab 08 — Collision vs Broadcast Domains

## Objective
Compare hub and switch behavior in a LAN.

## Topologies
- Hub-based network
- Switch-based network

## Steps
1. Built both topologies
2. Sent ICMP traffic
3. Observed traffic flow
4. Compared performance and behavior

## Key Concepts
- Collision Domain
- Broadcast Domain
- Network Segmentation

## Results
Switch reduces collisions and improves network efficiency.
